# Interview on HTML Basics

1. What does HTML stand for?
ans: HTML stands for HyperText Markup Language

2. What is the purpose of the <!DOCTYPE html> declaration?
ans: the <!DOCTYPE html> tells  the browser of the use of HTML 5 version

3. What is the difference between <head> and <body>?
ans: the <head> tag contains information that do not display directly on the web page, such as metadata, title, links etc whilest
the <body> tag contains information that is seen on the webpage such as paragraphs, sections, etc.

4. What does “semantic HTML” mean?
ans: semantic HTML means HTML tags or elements that provide meaning to both the browser and developers, 
and are crucial for accessibility and search engine optimization (SEO). 
5. Name three semantic HTML5 elements.
ans: <header>, <section>, <article>

6. What is the purpose of the <meta charset="UTF-8"> tag?
ans: the <meta charset="UTF-8"> tag is for character encoding

7. Is HTML a programming language? Explain briefly.
ans: HTML is not a programming language but a markup language 

8. What is the correct way to create a hyperlink in HTML?
ans: a hyperlink is correctly created using the anchor tag element that is <a href=""></a>

9. What is the purpose of the <alt> attribute on an <img> tag?
ans: the <alt> attribute on the <img> tag is used to add a descriptive text of what the image is, 
this is helpfull for accessibility reasons such as screen readers, or instants where image is not rendered.

14. What does the <title> tag do?

it is the name seen when searching for the website on the internet

22. What’s the difference between block-level and inline elements?
ans: block-level elements always starts on a new line and takes up the full width of it container 
but inline element flows along with its surrounding elements and only take up relevant space.

23. Which tag is used to group multiple form fields together?

24. What HTML element is used to define emphasized text?
ans: <em></em>

25. What’s the purpose of the <footer> element?
