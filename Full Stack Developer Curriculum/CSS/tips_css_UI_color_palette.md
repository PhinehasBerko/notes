# How to pick up color for UI

1. Must have one primary color to create attraction for main action of focus (essence)
2. and for everything else use a neutral color like white blacks and grays
3. use supporting or secondary colors for expressions that complement main action on a page (illustrations), they Must not be random.
4. use tint and shades for primary colors for "states" of an action.; tint is a lighter version of your base (primary) color and shades are the darker version.
