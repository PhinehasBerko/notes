# Docker Crash Course

# What is Docker?
- Docker is a virtualization software that makes developing and deploying applications much easier by 
  packaging application and  all the necessary dependencies, configurations, system tools and runtime 
  into a container.

  - A standardized unit that put together everything an application needs to run

  - making it a  portable artifact, easily shared and distributed
